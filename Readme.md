## Ejecutar en desarrollo

1. Clonar el repositorio
2. Instalar Dependencias
```
npm install
```
3. Configurar variables de entorno en archivo .env siguiendo ejemplo de archivo .env.example


5. Ejecutar Servidor local
```
npm run dev
```
## Stack de Desarrollo

1. Node.js
2. Javascript
3. Sequelize
4. MySQL
5. Express
